# Sentiment-Analysis-IMDB-NLP-
In this project I did as classical NLP + RNN job , which is to build a sentiment analysis deep learning model . For this I chose one of the popluar IMDB datasets and by doing the prominent NLP tasks and by applying RNN architecture ( using Pytorch ) I achieved an accuracy of around 84 % . 

I could make some changes by applying word2vec and LSTM , which I will do in the near future .
